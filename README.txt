=====================================
  부루마블 보드게임 사용 설명서
=====================================

■ 설치 방법

  Windows:
    1. install.bat 파일을 더블클릭하여 설치 확인
    2. run.bat 파일을 더블클릭하여 게임 실행

  Linux/Mac:
    1. 터미널에서 chmod +x install.sh run.sh 실행
    2. ./install.sh 로 설치 확인
    3. ./run.sh 로 게임 실행


■ 게임 방법

  1. 게임 시작 후 좌측 [설정] 버튼으로 게임 설정
     - 플레이어 수, 칸 개수, 문제 칸 위치, 주사위 최대값, 목표 바퀴 등

  2. [주사위 굴리기] 버튼을 눌러 주사위를 굴림

  3. 말 이동 방법 선택:
     - [자동 이동]: 주사위 결과만큼 자동으로 이동
     - [수동 이동 모드]: 보드판의 원하는 칸을 클릭하여 직접 이동

  4. 문제 칸(빨간색)에 도착하면 [문제 출력] 버튼이 활성화됨
     - 버튼을 눌러 문제 확인
     - [정답 보기] 버튼으로 정답 확인 가능

  5. [다음 턴] 버튼으로 다음 플레이어에게 턴 넘기기

  6. 목표 바퀴를 설정했다면, 해당 바퀴를 먼저 완주한 플레이어가 승리


■ 보드판 칸 종류

  금색 (0번 칸)  = 출발 칸
  빨간색         = 문제 칸
  흰색           = 일반 칸


■ 문제 파일 편집 방법

  questions.txt 파일을 메모장으로 열어서 편집합니다.

  작성 형식:

    문제: 여기에 문제를 적으세요
    정답: 여기에 정답을 적으세요

    문제: 두 번째 문제
    정답: 두 번째 정답

  규칙:
  - "문제: " 으로 시작하는 줄에 문제 작성
  - "정답: " 으로 시작하는 줄에 정답 작성
  - 문제와 문제 사이에 빈 줄 하나
  - 저장할 때 인코딩을 UTF-8로 선택
    (메모장: 파일 → 다른 이름으로 저장 → 인코딩: UTF-8)


■ 파일 목록

  main.py        - 메인 게임 프로그램
  questions.txt  - 문제 파일 (직접 편집 가능)
  install.bat    - Windows 설치 확인 스크립트
  install.sh     - Linux/Mac 설치 확인 스크립트
  run.bat        - Windows 실행 스크립트
  run.sh         - Linux/Mac 실행 스크립트
  README.txt     - 이 설명서
